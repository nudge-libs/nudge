Initial beta release of Nudge - a type-safe prompt builder for AI applications.

### @nudge/core
- Type-safe prompt builder with fluent API
- Support for system prompts, variables, and optional blocks
- Zero dependencies

### @nudge/cli  
- Generate type-safe prompt files from `.prompt.ts` files
- Watch mode for development
- Configurable via `nudge.config.json`
