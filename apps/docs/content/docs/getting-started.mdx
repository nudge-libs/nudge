## Quick Start

### 1. Create a prompt file

```ts title="src/summarizer.prompt.ts"
import { prompt } from "@nudge/core";

export const summarizerPrompt = prompt("summarizer", (p) =>
  p
    .persona("expert summarizer")
    .input("a block of text")
    .output("a concise summary")
    .do("preserve key facts")
    .constraint("keep under 3 paragraphs")
);
```

<Callout>
  Prompts must be in `*.prompt.ts` files for the CLI to find them.
</Callout>

### 2. Configure AI

```json title="nudge.config.json"
{
  "ai": {
    "provider": "openrouter",
    "apiKeyEnvVar": "OPENROUTER_API_KEY",
    "model": "anthropic/claude-sonnet-4"
  }
}
```

### 3. Generate

```bash
npx @nudge/cli
```

This creates `src/prompts.gen.ts` with your AI-generated system prompts.

### 4. Use it

```ts title="src/index.ts"
import "./prompts.gen";
import { summarizerPrompt } from "./summarizer.prompt";

console.log(summarizerPrompt.toString()); // â†’ AI-generated prompt
```
